
# Neural Network API

Trabalho para a disciplina de **Aprendizado de Máquina**: desenvolvimento de uma biblioteca de redes neurais do zero (no estilo de TensorFlow / PyTorch / scikit-learn), usando somente `numpy` como dependência externa.

## Objetivo

Construir, entender e documentar os componentes fundamentais de uma biblioteca de redes neurais: camadas, funções de ativação, perda, otimizadores, pipeline de treinamento e utilitários para avaliação.

## Estrutura do repositório

- `api/`: API desenvolvida e eventuais componentes auxiliares (RandomSearch, por exemplo).
- `datasets/`: conjuntos de dados de treino, validação e teste, manipulados via arquivos de `notebooks/`.
- `notebooks/`: notebooks com etapas de obtenção, exploração, preparação, treinamento e demonstração do modelo.
- `imagens/`: imagens usadas neste README, nas explicações dos notebooks e eventuais gráficos plotados.

## Dependências

- Python 3.12+
- `numpy v2.4.0` (ver `requirements.txt`)

## Uso

Seção a ser futuramente desenvolvida com importação e blocos de código de exemplo.
